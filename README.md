# 🐝 Smart Resume Analyzer

An AI-powered Resume Analyzer developed using Python, Streamlit, NLP, and Machine Learning concepts.

## 🚀 Features

* Resume Upload (PDF Support)
* ATS Score Analysis
* Resume Scoring System
* Skill Extraction using NLP
* Missing Skills Detection
* Course Recommendations
* Admin Dashboard
* Downloadable PDF Reports
* Interactive Bee-Themed UI

---

## 🛠️ Technologies Used

* Python
* Streamlit
* spaCy NLP
* PyPDF2
* MySQL
* Machine Learning Concepts

---

## 📊 Project Workflow

1. User uploads resume
2. Resume text is extracted from PDF
3. NLP processes resume content
4. Skills and sections are identified
5. ATS score is generated
6. Missing skills are detected
7. Recommendations and report are displayed

## ▶️ How to Run

Install required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run App.py
```
